# nightRes

Questo programma in linguaggio C gestisce un sistema di prenotazione per una discoteca chiamato “NightRes”. Permette di amministrare tavoli, clienti e prenotazioni tramite un menu interattivo su terminale. L’utente può aggiungere, modificare, eliminare e visualizzare dati, controllare le prenotazioni della serata, gestire una lista d’attesa e visualizzare statistiche. Il sistema carica i dati da file all’avvio, consente di salvarli durante l’utilizzo e libera la memoria prima della chiusura del programma.
